# キッズセーフ 鯖江

鯖江市の地域安全マップを作成し、スマートフォンやPCで共有できるツールです。ExcelやNumbersで編集したCSVデータをアップロードすることで、地図の内容を更新できます。[キッズセーフ by Code for FUKUI](https://github.com/code4fukui/kidssafe/) を使って作られています。

## 機能
- ExcelやNumbersで作成したCSVデータを地図に反映できる
- スマートフォンのホーム画面に登録して簡単に利用できる
- 外国語表示にも対応

## 使い方
### データの更新方法
1. 変更したいデータ(例:aed.csv)をダウンロードする
2. Excelで編集する
3. [緯度経度地図](https://fukuno.jig.jp/app/map/latlng/#%E8%B6%8A%E5%89%8D%E5%B8%82)で位置情報を確認し、Geo3x3欄の文字列をコピーする
4. Excelで保存し、[./](./) フォルダにアップロードする
5. 1分ほど待つと更新される

### データ種の追加方法
1. [template.csv](template.csv)をダウンロードしExcelで開く
2. 2行目以降に地図に設定したい情報を入力する
3. ファイル名を分かりやすい英数字で保存する
4. [index.csv](index.csv)をダウンロードしExcelで開き、3で保存したファイル名とデータ種類名、アイコンファイル名を記述する
5. [./](./) フォルダにindex.csvと3で保存したファイルをアップロードする
6. 1分ほど待つと反映される

### アイコンの追加方法
1. 100x100ピクセル程度のPNG画像を用意し、半角英数字のファイル名で保存する
2. [icon](icon)フォルダにアップロードする
3. [index.csv](index.csv)や各データをダウンロードし、icon項目を更新する
4. 編集したファイルを[./](./) にアップロードする
5. 1分ほど待つと反映される
