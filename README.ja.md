# KidsSafe for Sabae

鯖江市向けのカスタマイズ可能なコミュニティマッピングツールです。このプロジェクトでは、シンプルなCSVファイルを使用して、住民がインタラクティブな地図を簡単に作成・共有できます。[Code for FUKUIのKidsSafe](https://github.com/code4fukui/kidssafe/)フレームワークをベースに構築されています。

**ライブデモ:** [https://code4fukui.github.io/kidssafe-sabae/](https://code4fukui.github.io/kidssafe-sabae/)

## 特徴

- **CSVベース:** CSVファイルを編集してアップロードするだけで、地図レイヤーを作成・更新できます。
- **カスタマイズ可能:** 新しいデータタイプ（例：AED、遊び場、飲食店など）やカスタムマップアイコンを簡単に追加できます。
- **モバイルフレンドリー:** すぐにアクセスできるよう、スマートフォンのホーム画面に地図を追加する手順を用意しています。
- **多言語対応:** インターフェースを他の言語に切り替えることができます。

## 地図データの更新方法

GitHubのインターフェースから直接地図を更新できます。変更は通常1分以内に反映されます。

*注意: 変更が反映されない場合は、キャッシュを回避するために、ブラウザのプライベート/シークレットウィンドウでページを再読み込みしてみてください。*

### 1. 既存のデータを更新する

1. 変更したいファイル（例: [`aed.csv`](aed.csv)）を開きます。
2. 「Download raw file」ボタンをクリックしてダウンロードします。
3. ExcelやNumbersなどの表計算ソフトでファイルを開き、編集します。
   - 位置情報を取得するには、[緯度経度マップ](https://fukuno.jig.jp/app/map/latlng/#%E8%B6%8A%E5%89%8D%E5%B8%82)を使用し、`Geo3x3`コードをコピーしてください。
4. 変更を保存します。
5. リポジトリのメインページに戻り、更新したファイルをファイルリストにドラッグ＆ドロップしてアップロードします。

### 2. 新しいデータタイプ（地図レイヤー）を追加する

1. [`template.csv`](template.csv) ファイルをダウンロードします。
2. ファイルを開き、2行目から新しいデータを追加します。列は必要なだけ追加できます。
3. ファイル名を分かりやすい半角英数字（例: `playgrounds.csv`）にし、「CSV UTF-8 (コンマ区切り)」形式で保存します。
4. [`index.csv`](index.csv) をダウンロードして開きます。
5. 新しい行を追加し、新しいファイル名、地図レイヤーの表示名、使用するアイコンのファイル名を記入します。
6. 新しいデータのCSVファイルと、更新した `index.csv` の両方をリポジトリのルートディレクトリにアップロードします。

### 3. 新しいアイコンを追加する

1. PNG画像（100x100ピクセル程度が最適）を作成し、シンプルな半角英数字のファイル名を付けます。
2. [`icon`](icon) フォルダを開きます。
3. 新しいアイコン画像をフォルダ内にドラッグ＆ドロップしてアップロードします。
4. [`index.csv`](index.csv) または各データファイルの `icon` 列を更新し、新しいアイコンのファイル名を指定します。
